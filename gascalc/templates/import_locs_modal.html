Импорт адресов

Для импорта адресов поддерживаются файлы со следующими расширениями: csv, xls, xlsx

Файл должен содержать следущую информацию: адрес, организация, категория организации и квота поездок.

Рекомендации по наполнению:

location: адрес организации. В адресе необходимо указать город, для корректного определения координат и последующего построения маршрутов.

organization: наименование организации, как должно быть в путевом листе

category: категория организации (Клиент/Поставщик/Собственная/Другое). Категории нужны для гибкого построения маршрутов. Категория "Собственная" определяет стартовые и конечные точки для маршрутов. В этой категории должна быть хотя бы одна организация

quota: квота поездок. Определяет максимальное число поездок по данному адресу в расчетный период. Для организаций категории "Собственная" можно указать не указывать.
