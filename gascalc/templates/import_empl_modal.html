Импорт сотрудников

Для импорта сотрудников поддерживаются файлы со следующими расширениями: csv, xls, xlsx

Файл должен содержать следущую информацию: имя, фамилия, отчество, время работы, время остановки на адресе и квота поездок.

Рекомендации по наполнению:

first_name: имя сотрудника

second_name: фамилия сотрудника

surname: отчество сотрудника

workday: рабочий день в часах

duration: продолжительность остановки (Быстро, Средне, Медленно)

quota: квота поездок. Определяет максимальное число поездок по данному адресу в расчетный период. Для организаций категории "Собственная" можно указать не указывать.
